# Contributors

This project would not exist without its contributors. Thank you! 💜

We use the [all-contributors](https://allcontributors.org/) specification to
recognize all kinds of contributions, not just code:

| Emoji | Contribution type   | Emoji | Contribution type |
| ----- | ------------------- | ----- | ----------------- |
| 💻    | Code                | 🐛    | Bug reports       |
| 📖    | Documentation       | 🤔    | Ideas & planning  |
| 🚇    | Infrastructure      | 🔍    | Reviewing PRs     |
| 🎨    | Design              | 📝    | Blog / articles   |
| 💬    | Answering questions | ✅    | Tests             |

## How to get added

After your first merged PR (or other contribution), a maintainer will run:

```bash
npx all-contributors add <your-github-username> <type>
```

or you can open a PR adding yourself to the table below. No contribution is too
small — typo fixes count as `📖 documentation`.

## Recognition beyond this file

- Every release notes in [`CHANGELOG.md`](./CHANGELOG.md) thank first-time
  contributors by name.
- Significant domain contributions (e.g., a new order type or indicator) are
  credited in the relevant docs and release announcement.
